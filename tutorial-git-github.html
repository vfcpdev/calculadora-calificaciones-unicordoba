🚀 El poder del control de versiones

Git no es solo un comando; es una red de seguridad para tu código. Imagina poder viajar en el tiempo cada vez que cometes un error. Eso es Git.

$ git --version
git version 2.39.0

En este tutorial aprenderás a manejar archivos, crear estructuras algorítmicas en JavaScript y enviar tus resultados al LMS CINTIA.

📁 Nivel 1: El Inicio del Viaje

Antes de empezar, Git necesita saber quién eres. Esto es vital para que cada cambio tenga tu firma.

Comando Clave: git init convierte cualquier carpeta en un repositorio inteligente.

📸 Nivel 2: Fotos del código (Commits)

Un 'commit' es como una fotografía del estado actual de tus archivos. Para llegar ahí, primero movemos los archivos al área de preparación (Staging Area).

$ git status
Untracked files: index.html, style.css

Tip: Siempre usa mensajes de commit descriptivos que expliquen qué hiciste.

🌿 Nivel 3: Dimensiones Paralelas (Ramas)

Las ramas te permiten trabajar en nuevas funciones sin romper la versión principal que ya funciona.

☁️ Nivel 4: La Nube (GitHub)

GitHub es el hosting para tus repositorios. Aquí es donde compartes tu código con el mundo y lo entregas en CINTIA.

⚙️ Algoritmos con JavaScript

Git controla las versiones, pero JavaScript da vida a tus ideas. Repasemos los pilares:

// Estructura de Decisión
if (edad >= 18) {
    console.log("Acceso Permitido");
} else {
    console.log("Acceso Denegado");
}

// Estructura Repetitiva
for (let i = 0; i < 5; i++) {
    console.log("Iteración: " + i);
}

Interacción DOM: JavaScript permite capturar eventos de HTML y cambiar estilos CSS en tiempo real, justo como este tutorial.

🏆 Reto Final: Proyecto CINTIA

Objetivo: Construir una "Calculadora de Calificaciones" interactiva.

  1. Crea un archivo index.html con un formulario.
  2. Aplica estilos con styles.css (usa Flexbox o Grid).
  3. Usa JavaScript para calcular el promedio y determinar si el estudiante aprueba usando if/else.
  4. Inicializa un repositorio Git local.
  5. Súbelo a un repositorio público en GitHub.
  6. Copia el enlace del repo y pégalo en la plataforma CINTIA.
